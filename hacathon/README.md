# Smart Appointment Prioritization System

A triage dashboard for healthcare facilities with automatic patient prioritization.

## Structure

- `backend/` - Express.js server with MongoDB
- `src/` - React frontend

## Setup

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
npm install
npm start
```

## Features

- Patient registration with symptoms
- Automatic severity scoring
- Queue management with waiting time estimates
- Real-time status updates
- Configurable service parameters
